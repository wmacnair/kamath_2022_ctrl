# kamath_2022_ctrl

This repo shows data from Kamath _et al._, 2022, processed with `scprocess`.

The samples used are:
- ctrl samples only
- NR2A-positive sorted samples only

Here is a link to the Pages output from this analysis:
https://wmacnair.github.io/kamath_2022_ctrl/

Here is a link to the paper:
[Single-cell genomic profiling of human dopamine neurons identifies a population that selectively degenerates in Parkinson’s disease](https://www.nature.com/articles/s41593-022-01061-1)
